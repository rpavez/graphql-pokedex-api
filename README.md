This is a very simple Pokedex backend API built with Node, Express, GraphQL, and Babel.
Fork Based on [http://davidandsuzi.com](http://davidandsuzi.com).

### Running locally

```
yarn
yarn start
Goto: http://localhost:3000/graphql
```

### Running composer

```
docker-compose up
Goto: http://localhost:3000/graphql
```

### Dependencies
MongoDB
